# AI Risk Intelligence Agent

## Energy Supply Chain Resilience Platform

A full-stack AI-powered web application and decision intelligence platform designed to monitor geopolitical and logistics risks, simulate energy supply chain disruptions, and generate explainable, actionable recommendations for import-dependent economies.

## Core Pipeline

Data Ingestion
→ AI Risk Intelligence
→ Scenario Simulation
→ Digital Twin
→ Procurement & SPR Optimization
→ Explainable AI Recommendations
→ Decision Execution

## Major Components

1. Data Ingestion & Backend
2. AI Risk Intelligence
3. Scenario Simulation & Digital Twin
4. Recommendations & Decision Intelligence

- Live Geopolitical Risk Monitoring

## Technology Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python + FastAPI
- Main Database: PostgreSQL + PostGIS
- Knowledge Graph: Neo4j
- RAG: ChromaDB
- Multi-Agent AI: LangGraph
- Simulation: Python, Pandas, NumPy
- Geospatial: Mapbox / Leaflet
- Visualization: Plotly / D3.js
- Cache: Redis (optional)
- ML: XGBoost / LightGBM (where appropriate)
